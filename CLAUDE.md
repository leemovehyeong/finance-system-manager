# 금융시스템매니저

VAN/POS 업무관리 앱. Next.js 14 + Tailwind + Supabase Pro + 카카오맵.

## 규칙
- 설계서: /docs/설계서.md
- 아키텍처: /docs/ARCHITECTURE.md
- 변경이력: /docs/CHANGELOG.md
- 에러기록: /docs/TROUBLESHOOTING.md
- 상수는 /lib/constants.ts만 수정. 하드코딩 금지.
- 수정 시 영향받는 모든 파일 함께 수정.
